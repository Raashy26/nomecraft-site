# nomecraft-site# NomeCraft
Ferramenta gratuita para gerar nomes e bios criativas para TikTok, Instagram, jogos e muito mais.

## Estrutura
- `/assets/` → CSS e JS
- `/ads/` → Scripts de monetização (Monetag)
- `/data/` → Dados de apoio (JSON)
- Páginas: index.html, gerador-nomes.html, gerador-bios.html, dicas.html, sobre.html

## Deploy
Publicar via **GitHub Pages**:
1. Settings → Pages
2. Source: `main` branch → `/ (root)`
